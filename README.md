# Benford-App

Benford-App is a Python API and a web page testing Benford's law.

## Local installation

```bash
git clone https://github.com/Demetriade/benford-app.git

cd api
py -m venv my_env
.\my_env\Scripts\activate
pip install -r requirements.txt

cd ../frontend
npm install
```

## Local execution (API)

```bash
cd api
.\my_env\Scripts\activate
python api.py
```

## Local execution (frontend)

```bash
cd frontend
npm start
```
